# refresh-count
A sample project to count total view by refreshing your page using node and docker

<h3> Start the project </h3>
<ul>
  <li>
    Clone the project to your local machine
  </li>
  <li style="color:blue;">
    docker-compose up --build
  </li>
  <li>
    Refresh page to check changes
  </li>
  </ul>
